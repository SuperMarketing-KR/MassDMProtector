** node.js 설치 필수 **

1. tokens.txt 에다가 구매한 토큰을 입력 (토큰, 이메일:비번:토큰 모두 지원)
2. config/config.js 수정
3. npm install 명령어로 필요한 모듈 설치
4. inviteToken.js로 낚시토큰 초대
5. index.js 실행
